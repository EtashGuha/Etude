IKernelLink.TypesetStandardForm Property

Gets or sets whether images created by the EvaluateToTypeset method should use Mathematica StandardForm (vs. TraditionalForm).

[Visual Basic]
Property TypesetStandardForm As Boolean
[C#]
bool TypesetStandardForm {get; set;}

Remarks

The default is true. False means to use TraditionalForm.
